# Байесовская оптимизация с ограничениями

## Аннотация
Данная работа посвящена сравнению четырех методов байесовской оптимизации для решения задач с ограничениями: 
CEI (Constrained Expected Improvement), штрафной метод, метод множителей Лагранжа и барьерный метод.
